How it works (Season Model)

  1. Sign in with Google → link your YouTube channel.
  2. Create a public playlist “AI Games Challenge — Season X”.
  3. Paste the provided verification token into playlist description.
  4. Publish videos to that playlist → leaderboard updates daily.
